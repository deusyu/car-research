### 核心层结论

| 车型 | 判断 |
|------|------|
| SU7 | 安全硬件达标，但品控和召回记录是减分项 |
| YU7 | 安全硬件宣称达标，但无第三方验证，可靠性完全未知 |
| Model Y | 安全认证权威，可靠性验证最充分，续航入门版偏短 |
| ET5/ET5T | **碰撞成绩最强（中欧双冠军）**，续航75kWh偏短但换电补偿，品牌财务风险是核心层以外的隐忧 |
| ES6 | Euro NCAP五星93%，换电消除续航焦虑，但品控投诉和财务风险同上 |
| 007GT | 安全设计有亮点（逃生设计好），但无独立碰撞成绩，上市1年激光雷达/NZP批量故障，品控投诉已显现 |

**七车都能安全到达B，但对"到达确定性"的信心不同。Model Y > 蔚来ET5/ES6 ≈ 极氪001（碰撞成绩好但品控差） > SU7 > 007GT > YU7（按验证程度+品控排序）。蔚来的额外风险在核心层之外——品牌能否持续运营。极氪的额外风险——品控投诉量极高+电池召回+起火记录。**
