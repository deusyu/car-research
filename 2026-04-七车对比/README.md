## 核心结论

**总分排名：SU7 (7.65) > 007GT (7.58) > ET5 (7.32) > 001 (7.15) > YU7 (7.13) > Model Y (7.05) > ES6 (6.83)**

排名随权重偏好翻转：
- 安全第一 → Model Y
- 家庭实用 → YU7
- 性价比 → 007GT / SU7
- 蔚来风险消除 → ET5
- 极度厌恶风险 → Model Y

极氪两车画像差异显著：
- **007GT**：20万级配置性价比之王（全系空悬+智驾+800V），但上市仅1年、品控故障已现、2026款即将发布存在"背刺"风险
- **001**：产品设计力强（碰撞双五星+空间最大+充电理论最快+影音最强），但品控口碑七车最差（投诉榜首+电池召回+8起起火）
