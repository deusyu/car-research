## 底盘与安全

### 底盘悬架

| 项目 | 规格 |
|------|------|
| 前悬架 | 双叉臂独立悬架（全铝合金摆臂） |
| 后悬架 | 五连杆独立悬架（下摆臂/上控制臂为铝合金） |
| 空气悬架 | 全系标配闭式空悬，5档高度调节，行程55mm |
| 减振系统 | 全系标配CCD电磁减振 + 智能魔毯 |
| 轮胎 | 米其林Pilot Sport EV静音胎 255/40 R20 |
| 制动 | Brembo前卡钳，379mm制动盘，100-0km/h 34.4m |
| 最大离地间隙 | 169mm（空悬最高） |
| 转向比 | 12.5:1 |

- **对比007轿车**：底盘刚性提升20-30%；后悬由多连杆升级为五连杆；标配空悬+CCD

### 动态驾驶评价

- **正面**：弯道极限高，空悬+CCD滤振柔韧，舒适与操控兼顾
- **批评**：转向偏轻电子味重、刹车初段空行程、2.2吨车重急变道侧倾明显、高速无框车门风噪

### 碰撞安全

**007GT暂无独立C-IASI成绩**。同平台极氪007（2024年C-IASI）：车内乘员/车外行人/辅助安全均为G+；维修经济性M/可维修性P。

007GT安全设计亮点：
- 行业首创电机下沉技术（碰撞时电机下移减少乘员舱侵入）
- G-AES通用障碍物自动避让（130km/h可触发）
- 车身高强度钢占比77.44%，28处1500MPa热成型钢
- 车顶承压近11吨（自重4.64倍）
