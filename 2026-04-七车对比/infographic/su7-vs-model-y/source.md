# 小米SU7焕新版 vs Tesla Model Y 购车决策对比

面向长辈阅读，风格要稳重专业。

## 板块一：销量趋势 - 市场用脚投票

- 小米SU7 2025年销量：25.82万辆，同比+200%，20万+纯电轿车销冠
- Tesla Model Y 2025年销量：42.53万辆，同比-16%，首次年度下滑
- 小米品牌全年41.18万辆，新势力增速最快
- 特斯拉中国全年~55万辆，增长见顶

## 板块二：安全性对比 - 两车都安全

- 碰撞测试：SU7 中保研C-IASI三项G+（最高） vs Model Y Euro NCAP五星+IIHS TSP+（最高）→ 持平
- 起火率：SU7约0.12/万，行业均值0.96/万，燃油车1.5/万 → 都远低于均值
- 气囊：SU7 9个 vs Model Y 8个
- 车身强度钢：SU7 2200MPa vs Model Y 未公布
- 逃生设计（关键差异）：SU7焕新版新增机械应急开门 vs Model Y断电后无法外部开门（已致全球15+人死亡）

## 板块三：智驾与泊车 - 北京停车刚需（重点突出）

- SU7智驾：300+城市NOA，全系免费
- Model Y智驾：FSD需额外花6.4万元购买，中国体验远不如美国
- SU7自动泊车：全系免费标配，支持极窄车位自动泊入泊出
- Model Y自动泊车：包含在FSD包中，需6.4万元
- 北京商场车位窄：SU7宽1963mm vs Model Y宽1920mm（含后视镜2129mm），两车尺寸接近
- 结论：在北京停车场景下，SU7的免费自动泊车是实实在在的刚需，省下的6.4万可以用好几年

## 板块四：价格 - 数学题不骗人

- SU7标准版 21.99万 vs Model Y后驱版 26.35万
- SU7便宜4.36万，但续航多127km（720 vs 593km）、智驾免费、气囊多1个
- 如果Model Y补齐智驾 = 26.35+6.4 = 32.75万，比SU7贵近11万
- 结论：21.99万买到的东西，Model Y需要花32.75万才能补齐

## 底部结论

Model Y是好车，但SU7焕新版在安全配置、智驾、续航、价格四个维度全面领先。这不是品牌偏好，是性价比的数学题。

数据来源：中保研C-IASI、Euro NCAP、IIHS、央视网/应急管理部、Tesla安全报告、证券时报
