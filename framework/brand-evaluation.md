# 品牌评估方法论

> 独立于单车档案的品牌级评估框架
> 产出品牌信任评分与品牌信号评分，作为决策矩阵的独立输入

---

## 第一性原理

车的本质是 **将人安全地从 A 带到 B。**

品牌的本质是 **降低购买决策中不确定性的累积信任信号。**

你无法亲自碰撞测试每辆车，无法预知 5 年后的质量表现，无法确定公司是否还在。品牌替你"担保"了这些未知。品牌越强，你为未知支付的风险溢价越低。

但品牌还有第二重作用：**它是社交信号的载体。** 同一个品牌在不同受众眼中传递不同的含义。品牌的社交价值不是品牌自身的属性，而是品牌与观察者之间的关系。

因此，品牌评估分为两个正交维度：

| 维度 | 方向 | 回答的问题 |
|------|------|-----------|
| **信任维度** | 品牌 → 买家 | 这个品牌降低了我多少决策不确定性？ |
| **信号维度** | 品牌 → 社会 | 这个品牌替我传递了多少社交信息？ |

---

## A. 信任维度（品牌 → 买家）

### 定义

信任维度衡量品牌作为"不确定性折扣券"的价值——同样一个未知问题（这车安全吗？出了问题有人管吗？），不同品牌给你的确定感不同。

### 五个信任子维度

| # | 子维度 | 评估什么 | 核心指标 | 数据来源 |
|---|--------|---------|---------|---------|
| 1 | 工程信任 | 我能相信它造的车是安全的吗？ | 造车年限、累计产量、平台代次与复用度、碰撞成绩一致性（跨车型） | 品牌官方历史、C-IASI/Euro NCAP 全系成绩 |
| 2 | 质控信任 | 同一款车的品质是否稳定？ | 千车故障率(PPH)、召回响应速度（发现→公告天数）、车质网投诉密度趋势 | J.D. Power IQS、车质网、市场监管总局 |
| 3 | 存续信任 | 5 年后这个品牌还在吗？ | 盈利/亏损趋势、现金储备/烧钱速度、母公司背书强度、市占率变化方向 | 财报、行业分析、股价趋势 |
| 4 | 服务信任 | 出了问题能兜住吗？ | 服务网点密度（每万辆保有量对应网点数）、危机响应模式、保修政策慷慨度 | 官方网络、投诉平台事件复盘 |
| 5 | 残值信任 | 市场认不认这个品牌？ | 1/3 年保值率中品牌归因比例、二手车流通周期、同平台不同品牌的残值差 | 二手车平台（瓜子、懂车帝）、保险定价 |

### 评分方法

每个子维度 1-10 分，评分规则：

| 分数段 | 含义 |
|--------|------|
| 9-10 | 行业标杆，无需担忧（如丰田的质控信任、BBA 的残值信任） |
| 7-8 | 高于平均，有据可查 |
| 5-6 | 行业平均水平 |
| 3-4 | 低于平均，存在可验证的问题 |
| 1-2 | 严重缺陷或无数据可评估 |

### 评分规范

- 每个分数必须附带理由和数据来源
- 未经第三方验证的品牌自述不给高分
- 新品牌（造车 < 3 年）工程信任上限为 6 分——时间是信任的原材料，无法跳过
- 存续信任仅对新势力/独立品牌适用；有强母公司背书的品牌（如极氪之于吉利）可获得母公司信用传递，但需打折（母公司不等于无条件兜底）
- 同一集团下的不同品牌分别评估（极氪 ≠ 吉利 ≠ 领克）

### 信任总分

```
信任总分 = Σ(子维度得分 × 子维度权重)
```

默认等权（各 20%）。可根据买家偏好调整：

| 买家类型 | 权重侧重 |
|---------|---------|
| 风险厌恶型 | 存续信任 40%，工程信任 30% |
| 实用主义型 | 质控信任 30%，服务信任 30% |
| 投资保值型 | 残值信任 40%，存续信任 30% |

## B. 信号维度（品牌 → 社会）

### 定义

信号维度衡量品牌作为"社交信息载体"的传播效能——它在多大范围内、多高一致性地传递你想传递的信号。

### 核心模型

品牌信号价值不是品牌的固有属性，而是 **品牌 × 受众** 的函数。

同一个品牌在不同人群眼中是完全不同的东西：

- 奔驰 E：一线高收入眼中"中产标配"，县城眼中"大老板"
- 蔚来：一线 EV 圈"中国豪华电动"，三线可能不认识
- 小米 SU7：科技圈"性价比之王"，传统商务圈"手机厂造车"

因此需要将品牌感知拆分为**客观层**（可调研）和**主观层**（买家输入）。

### B.1 认知图谱（客观层）

对每个品牌，调研其在 N 个受众切面中的**认知分**（1-10 分）。

#### 受众切面定义

| 切面 | 描述 | 调研方式 |
|------|------|---------|
| 一线高收入 | 家庭年收入 > 80 万，一线城市 | 社交媒体语境分析、高端社区讨论 |
| 一线中产 | 家庭年收入 30-80 万，一线城市 | 汽车论坛、知乎、小红书 |
| 二三线城市 | 二三线城市主流消费者 | 地方论坛、懂车帝评论区 |
| 四五线/县乡 | 县城及农村 | 短视频平台评论、线下认知调研 |
| 科技/互联网圈 | 科技行业从业者 | 即刻、V2EX、科技媒体 |
| 传统商务圈 | 体制内、传统行业、商务接待场景 | 商务用车偏好、政府采购名录 |

<!-- 切面可根据实际需求增减，但不少于 4 个 -->

#### 认知分评分标准

| 分数段 | 含义 |
|--------|------|
| 9-10 | 该人群中的标杆/默认选择（如 BBA 在传统商务圈） |
| 7-8 | 广泛认可，正面联想为主 |
| 5-6 | 知道但无明确好恶 |
| 3-4 | 知道但认知模糊或存在负面联想 |
| 1-2 | 基本不知道或强烈负面 |

#### 从认知图谱中提取的指标

```
最大公约数 = min(所有切面认知分)
```
所有人群的最低共识认知。最大公约数高 = 品牌在任何场景都不会"失灵"。

```
认知方差 = var(所有切面认知分)
```
方差越大，品牌的社交信号越不稳定，跨场景风险越高。

```
认知天花板 = max(所有切面认知分)
```
品牌在最优受众中的峰值表现。

### B.2 场景分布向量（主观层）

买家输入自己的社交场景权重分配。所有权重之和 = 1。

```
场景向量[买家] = {
  一线高收入:   w₁,
  一线中产:     w₂,
  二三线城市:   w₃,
  四五线/县乡:  w₄,
  科技/互联网圈: w₅,
  传统商务圈:   w₆,
}
```

#### 典型场景向量预设

| 买家画像 | 一线高收入 | 一线中产 | 二三线 | 县乡 | 科技圈 | 商务圈 |
|---------|-----------|---------|--------|------|--------|--------|
| 一线互联网从业者 | 0.1 | 0.3 | 0.1 | 0.1 | 0.3 | 0.1 |
| 二线城市公务员 | 0.05 | 0.2 | 0.3 | 0.15 | 0.05 | 0.25 |
| 跨场景商务人士 | 0.2 | 0.2 | 0.2 | 0.1 | 0.1 | 0.2 |
| 县城创业者 | 0.05 | 0.1 | 0.25 | 0.35 | 0.05 | 0.2 |

<!-- 买家也可自定义向量，预设仅供参考 -->

### B.3 信号得分计算

```
信号得分 = 场景向量 · 认知图谱ᵀ    // 加权点积
```

即：

```
信号得分 = Σ(场景权重ᵢ × 品牌在切面ᵢ中的认知分)
```

#### 辅助指标

| 指标 | 公式 | 含义 |
|------|------|------|
| 信号稳定性 | `1 / (1 + 认知方差)` | 方差越大越不稳定，跨场景越容易"失灵" |
| 信号底线 | `最大公约数` | 最差场景下的品牌认知下限 |
| 信号峰值 | `认知天花板` | 最优场景下的品牌表现上限 |
| 信号覆盖率 | `认知分 ≥ 5 的切面数 / 总切面数` | 品牌在多少比例的场景中"及格" |

## 品牌综合评分

```
品牌综合分 = α × 信任总分 + β × 信号总分
```

其中 α + β = 1。

| 买家类型 | α（信任） | β（信号） | 理由 |
|---------|----------|----------|------|
| 理性工具型 | 0.8 | 0.2 | 车是工具，品牌只要靠谱就行 |
| 均衡型（默认） | 0.6 | 0.4 | 信任为主，信号为辅 |
| 社交敏感型 | 0.4 | 0.6 | 品牌的社交价值是重要决策因素 |

## 与决策矩阵的集成

### 方式：独立列

品牌综合分作为决策矩阵的独立评分列，占总权重的一定比例。

原有五层权重需要重新分配以容纳品牌维度：

| 维度 | 原权重 | 调整后权重（建议） |
|------|--------|-------------------|
| 核心层 | 35% | 30% |
| 效率层 | 25% | 22% |
| 成本层 | 20% | 18% |
| 体验层 | 15% | 13% |
| 符号层 | 5% | 7% |
| **品牌层（新增）** | — | **10%** |

<!-- 品牌层从各层等比缩减权重而来，符号层略微增加是因为原符号层中的"品牌"子项已独立出去，剩余的设计语言/颜色/社区等内容可适当提权 -->

### 敏感性分析扩展

在原有敏感性分析场景基础上，增加品牌相关场景：

| 场景 | 品牌层权重 | 回答的问题 |
|------|-----------|-----------|
| 品牌无感型 | 0% | 如果完全不在意品牌，排名怎么变？ |
| 品牌驱动型 | 20% | 如果品牌是重要决策因素，排名怎么变？ |
| 纯信任型 | 10%（α=1.0） | 只看品牌信任不看社交信号 |
| 纯信号型 | 10%（β=1.0） | 只看社交信号不看品牌信任 |

## 数据规范

与整体框架一致：

- 每个数据点标注来源和年份
- 区分"官方数据" vs "第三方调研" vs "社交媒体观察"
- 认知图谱的评分需附带判断依据，不接受无来源的主观打分
- 品牌级数据（如 PPH）和车型级数据（如碰撞成绩）需明确区分层级

## 方法论边界

### 本框架能做的
- 系统化拆解品牌价值的构成
- 将"感觉这个品牌好"转化为可审计的评分
- 让不同买家基于同一份品牌数据得出不同但合理的结论

### 本框架不能做的
- 预测品牌未来走势（品牌力是滞后指标）
- 替代产品力评估（好品牌可以出烂车，烂品牌可以出好车）
- 消除主观性（信号维度的场景向量本质是主观偏好的显式化，不是客观化）

### 与负面信息档案的关系
- 品牌档案评估品牌整体"信任资本"
- 负面信息档案记录具体事件和风险
- 两者独立编写，但品牌档案的信任评分应参考负面档案中的事实
- 负面档案中的品牌级风险（如财务风险）会直接影响信任维度的"存续信任"评分
