# 搜索词模板库

> Skill 执行时 Agent 直接消费，按 Phase 分组

---

## Phase 1: 基础档案搜索词

每车执行 8-10 次 WebSearch，覆盖以下方向：

| 方向 | 搜索词模板 |
|------|-----------|
| 价格配置 | "{品牌} {车型} {年份}款 售价 配置 版本" |
| 电池续航 | "{品牌} {车型} CLTC续航 电池 kWh {年份}" |
| 充电速度 | "{品牌} {车型} 快充 10-80% 充电速度 实测" |
| 碰撞安全 | "{品牌} {车型} C-IASI Euro NCAP IIHS 碰撞测试 成绩" |
| 底盘悬架 | "{品牌} {车型} 悬架 底盘 试驾 评测" |
| 智驾系统 | "{品牌} {车型} 辅助驾驶 NOA 智驾 芯片 算力" |
| 空间尺寸 | "{品牌} {车型} 车身尺寸 后排空间 后备箱 实测" |
| 保养售后 | "{品牌} {车型} 保养费用 保值率 二手 售后" |
| 车机生态 | "{品牌} {车型} 车机 芯片 OTA 语音" |
| 国际评测 | "{brand} {model} {year} review test" |

### 优先信源

| 信源 | 类型 | 可信度 |
|------|------|--------|
| 官方配置表 / 官网 | 官方 | 高（价格、参数准确） |
| 新出行、42号车库 | 深度评测 | 高 |
| 汽车之家、懂车帝 | 综合 | 中高 |
| Autocar、Car and Driver | 国际评测 | 高（补充海外测试数据） |
| 知乎长文 | 用户深度 | 中（需甄别） |
| 论坛 / 贴吧 | 用户反馈 | 低（仅作参考） |

## Phase 2: 负面信息搜索词

每车/品牌执行 8-10 次 WebSearch，**只搜负面**：

| 方向 | 搜索词模板 |
|------|-----------|
| 安全事故 | "{品牌} {车型} 事故 起火 自燃 碰撞 死亡 {年份}" |
| 品控投诉 | "{品牌} {车型} 质量问题 投诉 异响 故障 车质网" |
| 召回记录 | "{品牌} {车型} 召回 市场监管总局 {年份}" |
| 智驾事故 | "{品牌} 辅助驾驶 事故 失灵 追尾 {年份}" |
| 售后维权 | "{品牌} 售后 维权 退车 拒保 {年份}" |
| 续航虚标 | "{品牌} 续航虚标 冬季衰减 实际续航 打折" |
| 财务风险 | "{品牌} 亏损 裁员 资金链 破产 {年份}" |
| 降价维权 | "{品牌} 降价 老车主 维权 补偿" |
| 黑猫投诉 | "黑猫投诉 {品牌} {车型}" |
| 车质网 | "车质网 {品牌} {车型} 投诉排行" |

### 负面信息验证标注规则

| 状态 | 标准 |
|------|------|
| `已证实（多源验证）` | ≥2家权威媒体报道 |
| `已证实（单源）` | 1家媒体但有实证（图片/视频/文件） |
| `待验证` | 用户爆料、论坛信息、无实证 |
| `厂商已回应` | 附厂商回应内容 |
