### 碰撞测试成绩

#### Euro NCAP（2024年测试，五星评级）

| 测试项目 | 得分 |
|---------|------|
| 成人乘员保护 | 89% |
| 儿童乘员保护 | 88% |
| 行人/弱势道路使用者保护 | 84% |
| 安全辅助系统 | 83% |
| **综合评级** | **五星** |

详细表现：
- AEB系统对车辆和行人/骑行者反应表现优异，接近满分
- 侧面碰撞：配备驾驶座中央气囊，前排驾驶员和乘客获得同等保护
- 前座和头枕在追尾场景下提供良好的颈椎保护
- 具备碰撞后自动刹车功能，防止二次碰撞
- 具备eCall系统，碰撞后自动报警
- 涉水安全：车门在断电2分钟内可打开，电动车窗在断电后仍可短时运行
- 小扣分项：对"开门撞"（dooring）场景的骑行者警告仅为"adequate"
- 行人头部保护中，A柱区域评价为"poor"

> **注意**：Euro NCAP测试的是欧洲版极氪001，与国内版基本同源，但配置可能有细微差异。极氪X也获得了Euro NCAP五星，并获评2024年度最安全小型车。

## 智驾系统

### 硬件配置

| 项目 | 规格 |
|------|------|
| 智驾方案 | 千里浩瀚H7 |
| 主芯片 | NVIDIA DRIVE Thor-U |
| 算力 | 700 TOPS（FP8精度） |
| 域控制器 | 极氪自研 ADCU Max |
| 激光雷达 | 1颗192线激光雷达（全系标配） |
| 摄像头 | 11颗车外摄像头 |
| 超声波雷达 | 12颗 |
| 传感器总计 | 31个感知单元（含激光雷达） |

> **对比2024/2025款**：2025款搭载英伟达Orin-X双芯片（算力508 TOPS），2026款升级至单颗Thor-U（700 TOPS），架构更优、功耗更低。

### Thor-U芯片技术优势

- 相比Orin-X：L3缓存从4MB提升至16MB（4倍）
- 引入Transformer Engine和硬件级稀疏计算支持
- 架构、算力、成本优化均优于前代
- 极氪自研计算平台提升120%认知准确度、21%预判准确性，减少8%碰撞风险

| 城市NOA | 城市道路领航辅助（自动处理红绿灯、交叉口、行人穿行） |
| HPA记忆泊车 | 学习并记忆常用泊车路线，一键泊入 |
| 车位到车位 | 从出发车位到目的地车位全程智驾 |
| APA自动泊车 | 自动识别车位并泊入 |
| L2级辅助驾驶 | 标配，可OTA升级 |
