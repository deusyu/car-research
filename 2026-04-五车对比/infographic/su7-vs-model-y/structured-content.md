# 小米SU7焕新版 vs Tesla Model Y 购车决策对比

## Overview
通过销量、安全、智驾泊车、价格四个维度的硬数据对比，展示小米SU7焕新版相比Tesla Model Y的全面性价比优势。面向长辈的理性决策参考。

## Learning Objectives
The viewer will understand:
1. 市场趋势已反转：SU7增速+200% vs Model Y首次下滑-16%
2. 两车安全性均高于行业均值，SU7在逃生设计上有关键优势
3. SU7免费智驾+自动泊车是北京场景刚需，Model Y需额外6.4万
4. 同等配置下SU7比Model Y便宜近11万
